Welcome to MusicNet!

Social Network Name

Our social network is called "MusicNet," built using React.js for the frontend and a Python server for the backend. We use MySQL to manage and store the data.

Additional Pages

We have added two additional pages:

  • Piano: A virtual piano where users can play their favorite tunes.
  • Survey: A page where users can vote for the best band in the world.

What Was Hard to Do?

One of the most challenging tasks was setting up cookies to enhance user experience and maintain session information securely.

Server Routes

Here are the different server routes that our app supports:

  • /Piano: Access the virtual piano.
  • /Survey: Participate in the "Best Band in the World" survey.
  • /Welcome: Welcome page for new users.
  • /FollowingUsers: View users you are following.
  • /Marketplace: Explore and shop in the marketplace.
  • /New: Create and interact with posts and content.
